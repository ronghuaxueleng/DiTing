# 🧠 谛听 (DiTing) 思维导图快捷键指南

欢迎使用 AI 笔记思维导图组件的键盘导航指南。您无需使用鼠标，即可完全遍历、整理和查看您的转录笔记内容！

> [!NOTE] 
> 提示：在使用这些快捷键之前，请**在思维导图面板内任一位置点击一下**，以激活该区域的键盘焦点。

## 🗺️ 节点导航 (方向键)

使用方向键在思维导图节点间流畅穿梭：

| 键位 | 操作 | 详细说明 |
| :--- | :--- | :--- |
| **`向右`** | 进入子节点 | 充当**深度优先搜索 (DFS) 的触发器**。一直按住向右键，可以自动展开隐藏的子节点，在遍历完当前分支后自动跳出并进入下一个分支，按照阅读顺序遍历图中所有节点。 |
| **`向左`** | 返回父节点 | 焦点立刻跳回当前选中节点的父节点。 |
| **`向上`** | 上一个兄弟节点 | 移动到当前节点正上方的相邻兄弟节点。如果你在当前同级列表的最上方，它会自动“跳跃”到父级、甚至爷爷级的上一个兄弟分支继续遍历。 |
| **`向下`** | 下一个兄弟节点 | 移动到当前节点正下方的相邻兄弟节点。如果你在同级列表的最下方，它会自动跳转到下一个相邻层级的可用分支。 |

## 🕹️ 交互与视图控制

控制焦点的显示与整体排版：

| 键位 | 操作 | 说明 |
| :--- | :--- | :--- |
| **`空格`** | 展开或折叠 | 展开或收起当前高亮节点的所有子节点。 |
| **`回车`** | 居中与聚焦放大 | 激活**专注模式**。强行将当前选中的节点拉到屏幕正中央，并自动将其放大到 100% 的最佳阅读比例。 |
| **`Ctrl` + `回车`** / **`Cmd` + `回车`** | 自适应缩放 | 一键缩放并重新居中，以在当前视口内完整预览**整个**思维导图全貌。 |

## 🔍 缩放与层级深度控制

控制您正在查看的范围和缩放比例：

| 快捷键 | 操作 | 说明 |
| :--- | :--- | :--- |
| **`Ctrl` + `向上`** | 放大视口 (+) | 放大思维导图，并以您当前聚焦的节点为缩放中心（锚点），防止节点跑偏。 |
| **`Ctrl` + `向下`** | 缩小视口 (-) | 缩小思维导图，同样以当前聚焦的节点为缩放中心。 |
| **`1` - `9` 数字键** | 切换全局展示层级 | 瞬间设定思维导图全局最多展示几层节点。*如果再次按下已经被设定的相同数字，系统会强行将所有被手动展开过的零碎节点收拢，让导图瞬间恢复整洁。* |

> [!TIP]
> **智能边缘平移运作中:** 提示：在您使用方向键浏览节点时，只有当节点靠近屏幕边缘 20% 的区域时，整个画布才会发生平移。这大幅减少了快速遍历节点时引发的画面剧烈震颤与晕眩感！
